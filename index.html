Архитектура клиента

Архитектура клиентской части веб-приложения обычно состоит из трёх основных компонентов:

HTML

Основная разметка, которая
определяет структуру
веб-страницы.

CSS

Стиль, который определяет,
как элементы HTML будут
выглядеть на странице.

JS

Скриптовый язык, который
добавляет интерактивность
на страницу.

HTML Элементы и Атрибуты

(A) (/a)

Это элемент якоря (anchor), который используется для создания ссылок. Например:

href

Это атрибут, который указывает целевой URL (адрес) ссылки.
Он может содержать как абсолютные, так и относительные пути.

alt

Этот атрибут обычно используется в теге img, чтобы предоставить текстовое
описание изображения. Это важно для доступности и SEO.

src

Атрибут src указывает путь к ресурсу, например, изображению или видео. Это говорит браузеру, где найти файл.

class

Атрибут class используется для присвоения CSS классов элементу, позволяя его стилизовать через CSS.
